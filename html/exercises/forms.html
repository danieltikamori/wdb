Forms!

Used to send data to somewhere, like servers or another page.

action= attribute in form tag is used to specify where to send the query/data.

label is VERY important to have because associates with the input. Always for= attribute must be the same of id= attribute of input.

The name attribute specifies the name of an <input> element.

The name attribute is used to reference elements in a JavaScript, or to reference form data after a form is submitted.

Note: Only form elements with a name attribute will have their values passed when submitting a form.
