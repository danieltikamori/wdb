Image tag

To insert images, use just img tag, no closing tag, no content between.

img src="" is the standard. Avoid using attributes like width inline, use CSS.

If no image was found, use alt attribute to show the text instead. Also is useful to screen readers and SEO.

The tag use is: img src="image.jpg" alt="Image description here, what is the image."

Comment tag

Shortcut to comment: select the text, then [CTRL + /] to convert in comment quickly.

or use !-- comment text -- between<>.
