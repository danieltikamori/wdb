Tables - for data

Used to organizing data, not a good practice use to format pages.

Create table using <table>, <tr>to create a row, <th> to create a header and <td> to create a data cell.

Example:

Name Age Gender Graduate
Arthur 34 Male No
Anna 25 Female Yes
Table Footer

There's also thead, tbody and tfoot to make more readable.

rowspan and colspan

Make a cell occupy a determined number of spaces, row or column.

Example:

Name Age Gender Graduation AWS Certification
Associate Bachelor
Arthur 34 Male No Yes Yes
Anna 25 Female Yes No Yes
Table Footer Not much used But good to know
