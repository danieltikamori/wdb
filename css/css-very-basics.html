text-align

Text alignment. right, center, left. Work only on the content.

font-weight

numbers 100 to 900. 400 is normal, 700 bold. bold, normal, etc.

text-decoration

Controls appearance of decorative lines. Shorthand of text-decoration-line, text-decoration-color, text-decoration-style anb the newer ...-thickness.

none, underline, underline dotted, underline dotted red, green wavy underline, underline overline, line-through, etc.

line-height

Height of a line box. Used to set distances between lines of text.

normal, 2.5, 3em, 150%, 32px, etc.

letter-spacing

Horizontal spacing between text characters.

normal, .2rem, 1px, -2px, etc.

font-size

Absolute units like pixels aren't recommended for responsive websites. Use relative like %.
