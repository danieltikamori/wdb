Most used CSS Selectors

More info: https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Selectors

* Universal selector

Used at the top of the css file.

Element selector

h1 {},button {}, h1,h2{} (selects all h1 and h2 elements)

# id selector

#idname {}

. class selector

.classname {}

  Descendant selector

element1 element 2 {} (e.g. li a {}) - select all <a>'s that are nested inside an <li>.

.class1 span a - select only <a> inside span inside class1

+ Adjacent selector

Selects only the immediately adjacent element, not Descendant. h1 + p - only paragraphs immediately preceded by h1.

> Direct child selector

Select only the element that are direct children of another element. div > li - only li direct children of divs.

Element[Attribute] selector

input[type=text] {} - only attribute type text on input element.

a[href*="google"] {} - <a> elements with href containing "google".

a[href$=".org"] {} - <a> elements with href ending ".org".

a[class~="logo"] {} - <a> elements whose class attribute contains the word "logo". [attr~=value] Represents elements with an attribute name of attr whose value is a whitespace-separated list of words, one of which is exactly value.

a[href*="insensitive" i] {} - <a> elements with href containing case insensitive "insensitive".

a[href*="CasE" s] {} - <a> elements with href containing case sensitive "CasE".

Pseudo Classes

Keyword added to a selector that specifies a special state of the selected element(s).

More info: https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Selectors#pseudo-classes_and_pseudo-elements

Starts with ":"

:hover

a:hover - selects any <a> element when hovered.

button:active - selects active(being clicked) button.

.post:nth-of-type(3n) - select every 3rd, 6th, 9th so on class post element.

.even:nth-of-type(even) - selects only even element with "even" class.

Pseudo Elements

Starts with "::" + keyword added to a selector that lets you style a specific part of the selected element(s). For example, ::first-line can be used to change the font of the first line of a paragraph.

p:selection - selects selected section of paragraph.
